
# Swirl Cursor Effect


## Run Locally

Clone the project

```bash
  git clone https://github.com/ashishnallana/swirl-cursor-effect.git
```

Go to the project directory

```bash
  cd swirl-cursor-effect
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm run dev
```
